# EatDay Landing Page (React)

High-End Glassmorphism & Minimal High-Tech 디자인 컨셉의 랜딩 페이지입니다.

## Tech Stack
- React 18
- Tailwind CSS
- Framer Motion

## 실행 방법

```bash
cd app
npm install
npm run dev
```

브라우저에서 http://localhost:3000 접속 (포트 사용 중이면 터미널에 표시된 URL 사용)

## 빌드

```bash
npm run build
```

## 주요 기능
- Glassmorphism UI
- Mesh gradient + noise 배경
- Hero: 3D avatar, Floating animation, Mouse parallax, Data cards
- Bento Grid: 4개 핵심 기능 (아이콘 애니메이션 포함)
- Scroll: Reveal 효과, Sticky Problem vs Solution split-screen
- Typography: Text-Glow on key metrics
