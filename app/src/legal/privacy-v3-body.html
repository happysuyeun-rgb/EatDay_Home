제4조 (개인정보의 제3자 제공)

회사는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만 다음의 경우는 예외로 합니다.

  1. 이용자가 사전에 동의한 경우
  2. 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우
제5조 (개인정보 처리위탁 및 국외 이전)

회사는 서비스 제공을 위해 다음 업체에 개인정보 처리를 위탁하고 있습니다. 아래 업체는 모두 해외에 서버를 두고 있어 개인정보의 국외 이전이 발생합니다.

수탁 업체위탁 업무이전 국가이전 항목보유 기간
Supabase Inc. 데이터베이스 저장·관리, 인증 서비스 미국 회원 정보, 식단 기록, 게시물 등 전체 위탁 계약 종료 시
Vercel Inc. 웹 서비스 호스팅, CDN 미국 접속 로그, IP 주소 위탁 계약 종료 시
Anthropic PBC 생성형 AI 메시지 생성 (Claude API) 미국 닉네임, 식단 요약 정보, 연속 기록일 API 처리 후 즉시 파기
Google LLC 음식 이미지 인식 (Gemini API) 미국 식단 사진 이미지 API 처리 후 즉시 파기
Kakao Corp. 지도 서비스 (카카오맵 SDK) 대한민국 GPS 위치, 검색어 카카오 정책에 따름
위탁 업체는 서비스 제공 목적 이외의 용도로 개인정보를 처리하지 않습니다.
제6조 (AI 서비스 이용 시 정보 처리)

잇데이는 다음과 같이 두 가지 AI API를 활용합니다.

AI 서비스제공 업체처리 목적전송 정보서버 위치
음식 이미지 인식 Google LLC (Gemini API) 촬영한 음식 사진에서 음식명 추출 식단 사진 이미지 미국
개인화 식단 피드백 Anthropic PBC (Claude API) 식단 분석 메시지 생성, 영양소 추정 닉네임, 식단 요약 정보, 연속 기록일 미국
  1. 전송된 정보는 각 API 처리 후 즉시 파기되며, 각 업체의 개인정보 처리방침이 별도로 적용됩니다.
  2. AI 이미지 인식 시 전송된 식단 사진은 음식명 추출 목적 외 사용되지 않습니다.
  3. AI가 제공하는 모든 식단 분석·메시지는 참고용 정보이며, 의료·영양 전문가의 진단·처방을 대체하지 않습니다.
AI 피드백 기능을 사용하지 않으려면 앱 설정에서 비활성화할 수 있습니다.
제7조 (위치정보 처리)
  1. 잇데이는 맛집 지도 기능 제공을 위해 기기의 GPS 위치 정보를 수집할 수 있습니다.
  2. 위치 정보는 주변 다이어트 맛집 추천 및 지도 표시 목적으로만 사용되며, 서버에 저장되지 않습니다.
  3. 위치 정보 제공을 거부하셔도 서비스 이용은 가능하나, 주변 맛집 추천 기능은 제한됩니다.
  4. 위치 권한은 기기 설정에서 언제든지 변경할 수 있습니다.
제8조 (건강정보 취급)
  1. 키·체중·목표 체중·식단 기록 등은 건강 상태를 추론할 수 있는 민감한 정보에 준하여 취급합니다.
  2. 회사는 건강 관련 정보에 대해 암호화된 통신(HTTPS), Row Level Security(RLS) 등 강화된 보호 조치를 적용합니다.
  3. 이용자는 본 서비스 가입 시 "건강 관련 정보(체중, 식습관 등)를 기반으로 서비스를 제공한다"는 내용에 명시적으로 동의하게 됩니다.
  4. 수집된 건강 관련 정보는 서비스 제공 이외의 목적(광고, 제3자 제공 등)으로 사용되지 않습니다.

자동화된 처리 안내: 잇데이는 이용자의 식단 기록과 목표 정보를 분석하여 AI 피드백, 리포트, 아바타 변화를 제공합니다. 이에 이의가 있는 경우 이메일로 문의해 주세요.
제10조 (개인정보 보호 조치)
  1. 암호화 통신: 전 구간 HTTPS 암호화 통신을 적용합니다.
  2. 접근 제한 (RLS): Supabase Row Level Security를 통해 본인의 데이터만 접근 가능합니다.
  3. 최소 권한 수집: 서비스 제공에 필요한 최소한의 정보만 수집합니다.
  4. 비밀번호 미저장: Google/Apple OAuth를 통해 비밀번호를 직접 저장하지 않습니다.
  5. 완전 탈퇴: 회원 탈퇴 시 Supabase Auth 삭제(Edge Function)를 통해 인증 정보를 포함한 모든 데이터가 삭제됩니다.
  6. 환경변수 관리: API 키 등 민감한 정보는 환경변수로만 관리되며 코드에 노출되지 않습니다.
  7. 클라우드 보안: Supabase, Vercel 등 서비스 제공 업체의 보안 인프라를 활용합니다.
제11조 (쿠키 및 로그 정보)
  1. 서비스는 세션 유지를 위해 로컬 스토리지 및 쿠키를 사용합니다.
  2. 서비스 품질 향상을 위해 앱 사용 로그(오류 정보, 사용 패턴)를 수집할 수 있습니다.
  3. 서비스는 Mixpanel을 통해 서비스 이용 통계(화면 방문, 기능 사용 빈도 등)를 익명으로 수집합니다. 수집된 정보는 서비스 개선 목적으로만 사용됩니다.
  4. 이용자는 기기 설정에서 쿠키 저장을 거부할 수 있으나, 일부 서비스 이용이 제한될 수 있습니다.
