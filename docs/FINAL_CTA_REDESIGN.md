# Final CTA 섹션 리디자인 전략

**목표**: 마무리를 차분하면서도 기억에 남고, CTA가 의도적으로 느껴지도록  
**톤**: 따뜻하고, 현실적이며, 조용히 동기 부여하는 — 프로모션 느낌 없음

---

## A. Stronger Korean CTA Copy Options

### H2 (메인 문장) — 2줄 에디토리얼

| 옵션 | 1줄 | 2줄 |
|------|-----|-----|
| **A (현행)** | 건강한 습관은 거창한 결심보다, | 가벼운 기록에서 시작됩니다. |
| **B (시작)** | 오늘부터 뭔가 달라지려면, | 가벼운 기록이면 됩니다. |
| **C (부담 해소)** | 거창한 계획은 필요 없어요. | 오늘 먹은 것부터 적어보세요. |
| **D (연속성)** | 건강한 습관은 | 작은 기록이 쌓일 때 생깁니다. |
| **E (직관)** | 결심보다 기록이 먼저예요. | 오늘 한 끼부터 시작해보세요. |
| **F (따뜻함)** | 건강은 멀리 있는 게 아니에요. | 오늘 먹은 것을 기록하는 것부터. |

**추천**: A 또는 D — 짧고, “가벼움·연속성”이 브랜드 톤과 맞음.

---

### 서브카피 (CTA 직전)

| 옵션 | 문구 |
|------|------|
| A | 오늘 식단부터, 잇데이로 부담 없이 시작해보세요. |
| B | 오늘 한 끼만 기록해보세요. 거기서부터 이어집니다. |
| C | 검색 한 번이면 칼로리가 채워지니까, 부담 없어요. |
| D | 무료로 시작하고, 가볍게 이어가보세요. |
| E | 오늘부터. 한 끼부터. |
| F | 지금 당장이 아니어도 괜찮아요. 오늘 먹은 것부터. |

**추천**: A, B, 또는 E — 짧을수록 마무리에 맞음.

---

### CTA 버튼 문구

| 옵션 | 문구 |
|------|------|
| A | 무료로 시작하기 |
| B | 오늘부터 시작하기 |
| C | 가볍게 시작하기 |
| D | 기록 시작하기 |

**추천**: A — “무료”가 진입 장벽을 낮추는 데 유리.

---

### 칩 (선택 — 최소화 권장)

- **현재**: AI 식단 기록, 맛집 지도, 일일 리포트
- **변경 A**: 그대로 두되, 더 작고 은은하게
- **변경 B**: 1~2개로 축소 — “기록 · 피드백 · 맛집”
- **변경 C**: 칩 제거 — 헤드라인 + 서브 + 버튼만

**추천**: 칩은 2~3개, 작게 — 핵심 가치만 짧게 상기시키는 용도.

---

### Coli 보조 문구 (선택)

- **현재**: “함께 이어가요”
- **대안**: “기록할 때마다 함께해요” / “필요할 때마다 옆에 있어요” / 제거

## B. Final Section Layout

### 원칙

- **에디토리얼**: 요소 수 최소화
- **계층**: H2 > 서브 > [칩] > 버튼 > [Coli]
- **여백**: 넉넉한 세로·가로 패딩
- **중앙 정렬**: 단일 메시지에 집중

### 구조

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                         [넉넉한 여백]                         │
│                                                             │
│              건강한 습관은 거창한 결심보다,                     │
│              가벼운 기록에서 시작됩니다.                        │
│                                                             │
│              오늘 식단부터, 잇데이로 부담 없이 시작해보세요.      │
│                                                             │
│              [기록] [피드백] [맛집]   —  칩 (작게, 선택)        │
│                                                             │
│                    [ 무료로 시작하기 ]                        │
│                                                             │
│                    [Coli] 함께 이어가요   —  선택, 작게        │
│                                                             │
│                         [넉넉한 여백]                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 간격 권장

- **H2 ↔ 서브**: mb-8 ~ mb-10
- **서브 ↔ 칩**: mb-10 ~ mb-12 (칩 없으면 생략)
- **칩 ↔ 버튼**: mb-12 ~ mb-14
- **버튼 ↔ Coli**: mb-8 ~ mb-10
- **섹션 패딩**: py-36 md:py-52 (다른 섹션보다 15~20% 넉넉하게)

### 배경

- **scene-cta** 유지: 하단 radial gradient (leaf, opacity ~12%)
- **대안**: 상·하단 모두 아주 약한 gradient로 “마무리” 구간 느낌

## C. Motion Idea

### 진입 순서

| 요소 | 패턴 | 타이밍 |
|------|------|--------|
| H2 | opacity 0→1, y 32~40 → 0 | 0 ~ 0.8s |
| 서브 | opacity 0→1, y 16~20 → 0 | 0.15 ~ 0.7s |
| 칩 | opacity 0→1 (동시 또는 stagger 0.05s) | 0.25 ~ 0.6s |
| 버튼 | opacity 0→1, scale 0.98→1, y 12→0 | 0.35 ~ 0.75s |
| Coli | opacity 0→1 | 0.5 ~ 0.8s |

### Easing

- `[0.16, 1, 0.3, 1]` — 부드럽고 자연스러운 감속

### 호버

- **버튼**: `scale 1.02`, `y -2`, shadow 살짝 강화 (0.25s)
- **버튼 탭**: `scale 0.98` (0.1s)

### Ambient

- **없음** — 마무리 구간은 정적·차분하게
- 배경 gradient만 필요 시 아주 느리게 opacity 미세 변화 (선택)

### Reduced motion

- duration 0.01ms 또는 즉시 최종 상태

## D. Implementation Guidance

### 타이포

- **H2**: `text-3xl sm:text-4xl md:text-5xl lg:text-[3.5rem]` ~ `text-[4rem]`
- **line-height**: 1.1 ~ 1.15
- **tracking**: -0.03em
- **서브**: `text-lg md:text-xl`, `max-w-xl`

### 버튼

- **크기**: h-16, px-12 ~ px-14
- **스타일**: leaf 배경, bold, `shadow-[0_20px_50px_rgba(74,124,89,0.3)]`

### 칩

- **크기**: text-xs, px-3 py-2, rounded-full
- **색**: `text-[var(--accent-forest)]/80`, `border-[var(--accent-leaf)]/15`
- **개수**: 2~3개

### Coli

- **크기**: w-9 h-9, ColiAsset size="sm"
- **위치**: 버튼 아래, gap-8
- **문구**: “함께 이어가요” — text-xs, tertiary
