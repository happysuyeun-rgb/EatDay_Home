# Restaurant Map 섹션 리디자인 전략

**목표**: 맛집 지도를 EATDAY의 **시그니처 기능**처럼 느끼게 하기  
**반응**: “다른 식단 앱과 다르다”

---

## A. Redesign Concept

### 핵심 컨셉: “라이프스타일 유틸리티”

이 섹션은 **기능 소개**가 아니라 **실제로 내 주변에서 쓰는 순간**을 보여주는 장면으로 바꾼다.

- **공간**: 페이지 평균 섹션보다 **1.3~1.5배** 세로·가로 여유
- **대비**: 이전·다음 섹션과 **배경 톤·프레임**을 명확히 다르게
- **구성**: 지도가 **메인**, 카드와 필터는 **지원 역할**
- **인상**: “지도 앱 창”이 아니라 **실제로 유용한 도구**처럼 보이기

### 시그니처 포지셔닝

- “식단 앱인데 지도가 있다”가 아니라  
- “외식할 때도 식단 흐름을 잇는” **핵심 경험** 중 하나

## C. Visual Hierarchy Plan

### 섹션 크기

- **패딩**: `py-28 md:py-40` → `py-32 md:py-48` (다른 섹션보다 20~30% 넉넉하게)
- **배경**: `scene-map` 유지하되, 상·하단 **그라데이션 페이드**로 이전·다음 섹션과 구분
- **컨테이너**: `max-w-[1200px]` → `max-w-[1280px]` 또는 `1400px` (지도가 더 넓게)

### 지도 영역

- **높이**: `min(72vw,420px)` → **min(75vw, 480px)** / 데스크 `min(55vw, 620px)` ~ **680px**
- **프레임**: 다크 링 `#1a1f1c` 유지 — “지도 창” 느낌 강화
- **내부**: 그라데이션 + 그리드 + radial gradient — 현재보다 **약간 더 진한** 지도 톤 (location-aware 느낌)

### 상단: 카피 + 필터

```
┌─────────────────────────────────────────────────────────────────┐
│  [태그] 맛집 지도                                                 │
│  다이어트는 집 밖에서 무너지기 쉬우니까                             │
│  내 주변 저칼로리·고단백 식당을 바로 찾고, 메뉴 정보까지 확인하세요.   │
│                                                                  │
│  [저칼로리] [고단백] [비건]   —  필터를 지도 위 또는 옆에          │
└─────────────────────────────────────────────────────────────────┘
```

- 카피: 좌 정렬 또는 **전체 너비 중앙**
- 필터: 카피 아래 **또는** 지도 좌상단 오버레이 (지도 위에 떠 있는 UI 느낌)

### 지도 레이아웃 옵션

**옵션 A — 지도 풀블리드**
- 지도가 **섹션 전체 너비** (패딩만 적용)
- 카드: 지도 **우측 30%**에 세로 붙이거나, **하단** full-width strip

**옵션 B — 지도 + 사이드 카드**
- 지도 **60~65%** | 카드 **35~40%** (데스크)
- 카드가 **항상 보이는** 상태 — 핀 클릭 시 카드 내용만 교체

**옵션 C — 지도 중심, 카드 오버레이**
- 지도 **80%** 이상
- 카드: 지도 **우하단** 또는 **하단**에 오버레이 (반투명 백드롭)

**추천**: **옵션 A** — 지도가 최대한 넓게, 카드는 하단 또는 우측 오버레이. 라이프스타일 유틸리티 느낌에 맞음.

### 카드 구성

- **필수**: 식당명, 필터(고단백 등), 거리, 대표 메뉴 + 칼로리
- **선택**: 짧은 리뷰 스니펫 (“단백질 양 좋아요” 등) 또는 메뉴 미리보기 1~2줄
- **스타일**: 흰 박스가 아닌 **glass** (backdrop-blur) 또는 **soft shadow + border**로 지도와 시각적 계층 분리

### “근처 맥락” 표현

- 하단 바: `내 주변 500m` 또는 `현재 위치 기준 · 500m`
- 선택 시: `230m 도보 3분` (거리 → 도보 시간, 선택)

## D. Interaction / Motion Plan

### 진입

| 요소 | 패턴 | 타이밍 |
|------|------|--------|
| 카피 | fadeUp stagger | 0 ~ 0.5s |
| 필터 | fadeIn | 0.2 ~ 0.4s |
| 지도 컨테이너 | scale 0.97→1, opacity 0→1 | 0.15 ~ 0.9s |
| 지도 내부 | zoom 1.08→1 (soft) | 0.25 ~ 1.1s |
| 현재 위치 마커 | scale 0→1, opacity 0→1 | 0.4 ~ 0.7s |
| 식당 핀 | stagger pop (spring) | 0.5 ~ 1.0s |
| 카드 | fadeIn + slideUp | 0.6 ~ 0.9s |

### 필터 전환

- **클릭 시**: active 칩 — 배경색 + 그림자 transition (0.2s)
- **필터 적용 시**: 
  - 매칭되지 않는 핀: `opacity 0.3`, `scale 0.9` (0.25s)
  - 매칭 핀: `opacity 1`, `scale 1`
  - 선택된 첫 매칭으로 **카드 자동 전환** + AnimatePresence

### 핀 선택

- **클릭 시**: 
  - 선택 핀: pulse ring (기존 유지), `scale 1.1`
  - 카드: `key` 변경 → AnimatePresence `mode="wait"` 또는 `"popLayout"` — slide + crossfade
- **호버**: 비선택 핀 `scale 1.08`, 선택 핀 `scale 1.12`

### 카드

- **내용 전환**: `initial` opacity 0, y 12 / `animate` opacity 1, y 0 / `exit` opacity 0, y -8
- **선택**: 카드 자체 **expand** (선택 시 살짝 높이 증가) — 선택 사항

### 지도 zoom

- **진입 시**: `scale 1.06~1.1 → 1` (0.8~1.2s) — “지도가 펼쳐진다” 느낌
- **상호작용 시**: pinch/scroll zoom은 **비활성** (정적 데모이므로)

### Reduced motion

- zoom, stagger 제거
- opacity·색상 전환만 유지 (0.2s)
- 카드 전환 duration 단축

## E. Implementation Recommendations

### 섹션

- `py-32 md:py-48`
- `scene-map` 또는 커스텀: `linear-gradient` + `radial-gradient`로 상·하단 fade
- `max-w-[1280px]` 또는 `1400px` (지도가 더 넓게)

### 지도

- **높이**: `h-[min(75vw,480px)] md:h-[min(55vw,640px)] lg:h-[680px]`
- **프레임**: `rounded-2xl` 또는 `rounded-[1.5rem]`, `p-3`, `bg-[#1a1f1c]`, `shadow-[0_32px_80px_rgba(0,0,0,0.15)]`
- **내부**: gradient + grid + radial — opacity 미세 조정으로 “지도” 느낌 강화

### 필터

- **위치**: 카피 블록 아래 **또는** 지도 좌상단 (`absolute top-4 left-4`)
- **스타일**: `rounded-2xl`, `border-2`, active 시 `shadow-[0_8px_24px_rgba(74,124,89,0.25)]`

### 카드

- **데스크**: 지도 하단 `border-t` strip 또는 우측 `w-[320px]` 고정
- **스타일**: `backdrop-blur-md`, `bg-white/90`, `rounded-xl`, `p-6`
- **추가 데이터**: `reviewSnippet` 또는 `menuPreview` (1~2줄) — props로 전달

### 데이터 확장 예시

```js
// 현재
{ name, filter, distance, top, left, menu }

// 확장 (선택)
{ name, filter, distance, walkTime, top, left, menu, menuPreview?, reviewSnippet? }
```

### 체크리스트

- [ ] 섹션 패딩·max-width 확대
- [ ] 지도 높이 600px+ (desktop)
- [ ] 배경 톤으로 이전·다음 섹션과 대비
- [ ] 필터 chip 전환 smooth
- [ ] 핀 선택 시 pulse + 카드 AnimatePresence
- [ ] 지도 진입 zoom 1.08→1
- [ ] “현재 위치 기준 · 500m” 문구
- [ ] 선택: 리뷰 스니펫 또는 메뉴 미리보기
- [ ] `prefers-reduced-motion` 대응
