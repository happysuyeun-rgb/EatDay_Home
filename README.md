# EatDay 홈 (랜딩 페이지)

AI 식단 관리 웰니스-tech 서비스 **잇데이** 랜딩 페이지입니다.

## 구조

```
EatDay_Home/
├── app/              # React + Vite + Tailwind 랜딩 앱
├── docs/             # 섹션별 스펙 문서
├── vercel.json       # Vercel 배포 설정
└── README.md
```

## 실행

```bash
cd app
npm install
npm run dev
```

## 배포

Vercel에 연결 시 `vercel.json`에 따라 자동 빌드·배포됩니다.
Root Directory를 비워두면 루트 `vercel.json`이 적용됩니다.
